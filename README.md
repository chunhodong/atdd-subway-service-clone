### 지하철 노선도 프로젝트
[ATDD 강의](https://edu.nextstep.camp/c/R89PYi5H) 실습을 위한 지하철 노선도 애플리케이션의 클론 프로젝트입니다
<br><br>
미션수행기간 피드백받았는 부분을 개선하여 다시 구현해보았습니다

### Getting Started

### Install
#### npm 설치
```
cd frontend
npm install
```
> `frontend` 디렉토리에서 수행해야 합니다.

### Usage
#### webpack server 구동
```
npm run dev
```
#### application 구동
```
./gradlew bootRun
```
